NVision GUI documentation

Introduction

NVision Web Gui is the graphical user interface for the NVision project.
The app. makes use of the several technologies: HTML5, CSS2 and CSS3, SVG/VML and JavaScript (jQuery, Raphael.Js, BBQ, Ajax, JSON); most of the content is dynamically generated on the client side.
The communication between server and client is based on the exchange of JSON objects containing pure data; no html is generated on server side; this way the presentation layer logic is centralised on the client/browser.

The browser matrix the application is required to support is:

  • IE 7
  • IE 8

but it is regularly tested against:

  • IE 7
  • IE 8
  • IE 9
  • FireFox 3.5+
  • Google Chrome 10+

NVision object

The core of the application logic is hold by the JavaScript object named NVision (available in the global scope: window.NVision).
This object act as a hub for most of the js functions and it represents the application name-space.
